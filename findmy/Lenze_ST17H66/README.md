# FindMy firmware for Lenze st17h66
The source for `FindMy.hex` is here: https://github.com/biemster/st17h66_FindMy

## How to use:
`PySerial` needs to be installed, then just simply type
```
$ ./flash_st17h66.py <your base64 adv key>
```
